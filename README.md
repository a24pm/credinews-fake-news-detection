# CrediNews – Fake News Detection & Credibility Scoring System

CrediNews is a machine learning–based system that detects fake news articles and assigns a credibility score using NLP techniques, source reliability, and content consistency.

## Features
- Fake / Real news classification
- Credibility scoring (0–100)
- TF-IDF + Logistic Regression model
- Rule-based content and source analysis
- Command-line prediction interface

## Tech Stack
- Python
- scikit-learn
- NLTK
- Flask (optional)
- GitHub
